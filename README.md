# 🌐 Tradutor Offline - Português ↔ Espanhol

Um app mobile simples e rápido para traduzir entre português e espanhol **sem internet**.

## ✨ Recursos

- ✅ Tradução instantânea (PT → ES / ES → PT)
- ✅ Funciona **completamente offline**
- ✅ Histórico de traduções
- ✅ Botão copiar resultado
- ✅ Feedback tátil (vibração)
- ✅ Interface limpa e intuitiva
- ✅ **600+ palavras** em dicionário

## 📱 Como Instalar

### ⭐ OPÇÃO 1: Baixar APK Pronto (MAIS FÁCIL)

1. **Clique em "Releases"** no topo à direita deste repositório
2. **Baixe o arquivo `.apk`**
3. **Abra o arquivo no seu celular Android**
4. **Clique em "Instalar"**
5. **Pronto!** ✅ O app está no seu celular

---

### Opção 2: Testar com Expo Go

**Pré-requisitos:**
- Node.js instalado ([download](https://nodejs.org/))
- App "Expo Go" instalado no celular (Play Store)

**Passos:**

```bash
npm install
npm start
# Pressione 'a' para Android ou escaneie o QR code
```

## 📖 Como Usar

1. Escolha a direção (PT → ES ou ES → PT)
2. Digite uma palavra ou frase
3. Clique em "Traduzir"
4. Copie o resultado
5. Veja seu histórico
